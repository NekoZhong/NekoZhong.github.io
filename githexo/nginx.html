# Nginx

# Linux 环境下安装

centos8:

yum install nginx

安装过程中，如果中途停止，输入y继续，知道出现complete！则表示安装完成。

# 启动

nginx

# 修改配置文件

从默认页面找到需要修改的配置文件路径，通过命令打开nginx的配置文件

vim /etc/nginx/nginx.conf
  • 将user改成root
user root;
  • 修改端口为监听端口
listen       80;
  • 修改服务器为需要监听的服务器
server_name  IP地址/域名;
  • 指定访问的项目路径
    root         /root/seaset/;
    index        index.html；

# 重启nginx

nginx -s reload

# 其他指令

nginx -s stop 停止

nginx -s quit 安全退出
