# 虚拟机和云服务器

# 虚拟机

虚拟机相当于电脑中的电脑，不开机的状态下，不会占用内存的。 你可以将它当作你电脑中的一个应用来理解。

如果电脑关机，虚拟机也会跟着关机。

# 云服务器

云服务器是功能强大的物理或虚拟基础架构，用于执行应用和信息处理存储。

简单而言，你可以将它理解为云端的电脑。如果你将云服务器的系统装成你熟悉的Windows系统，基本上跟操作你自己的电脑一样。

理论上来讲，你甚至可以将自己的电脑改装成一台服务器。当然只是理论上，如果你只是局域网内使用，这样操作可行。但如果你需要让他人通过外网访问你的服务器，那就需要运营商分配公网IP和端口。

# 为什么要租用云服务器？可不可以不买？

如果你只是需要一个Linux的环境练习和学习，可以不买。虚拟机完全可以帮助你实现这一目的。

如果你希望写一些有趣的网站与大家分享，先可以考虑搭建免费的个人博客（比如我前几个视频介绍的GitHub和hexo搭建的博客）。当个人博客不能满足你开发网站的需求时，就可以考虑租用服务器。
